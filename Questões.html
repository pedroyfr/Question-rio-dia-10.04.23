Questões

1. Defina dispositivos móveis.

R. Dispositivos movéis são tecnologias digitais que permitem a mobilidade e o acesso a internet.

2. Qual o papel da internet no atual avanço tecnologico?

R.A internet possibilita hoje uma difusão rapida, atraves das novas tecnologias de informação, do conhecimento permitindo as empresas não so melhorar a sua eficiencia mas fundamentalmente oferecem novos produtos e serviços pelos quais os consumidores, atraves dos mecanismos de mercado, manifestam a sua preferencia.

3. Cite os principais tipos de dispositivos moveis

R. SmartPhones, Ultrabook, Notebook, Netbook, GPS e Tablets.

4. Para que serve o nivel de aplicação da Arquitetura Android?

R. Construida de forma modular, essa API de alto nivel implementa diversas funcionalidades como a criação de views, gerenciamento de recursos, notificações e alertas de usuario, navegação entre os aplicativos, etc.

5. O SO do Android é baseado no Linux. Qual seu papel na arquitetura do Android?

R. O Linux é a base da arquitetura do Android, fornecendo o kernel do sistema operacional que controla o acesso ao Hardware e gerencia os recursos do sistema.

6. Qual a principal característica do Android?

R.Android é um sistema operacional (SO) baseado no nucleo linux, projetado principalmente para dispositivos eletronicos moveis (como SmartPhones e tables) com tela sensivel ao toque ou interface de usuario baseada na manipulação direta.

7. Qual a diferença de um aplicativo nativo para um desenvolvido por terceiros?

R. Os aplicativos nativos são capazes de proporcionar aos usuarios uma experiencia mais intensa do que os hibridos. Porque eles são desenvolvidos especialmente para funcionar em determinado sistema operacional e oferecer melhores recursos de Hardware, facilitando a utilização de funções e recursos proprios do app. Existe tambem um aplicativo chamado web app que é bem diferente do aplicativo nativo porque ele não é um aplicativo real e não estara disponivel nas lojas dos sistemas operacionais.

8. Como o Android se diferencia de outros sistemas de dispositivos moveis?

R. O Android é um sistema operacional feito para diversas marcas, como HTC, Motorola, Samsung, etc. Ele é feito sob a tutela do open-source, o que permite que qualquer um modifique, seja o usuario ou companhias.

9. Como funciona o nivel de framework da arquitetura do Android?

R. Esta camada é desenvolvida quase toda em java, e faz a interface com as aplicações Android. Ela provê um conjunto de bibliotecas para acessar os diversos recursos do dispositivo como interface grafica, telefonia, localizador(GPS), banco de dados persistente, armazenamento no cartão SD, etc.

10. E o nivel de bibliotecas?

R. Existemformas diferentes de catalogação, mas a mais utilizada é a classificação Decimal de Dewey - CDD, para os íntimos - comum na maioria.
